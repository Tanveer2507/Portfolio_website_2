HTML Developer (2010-2012)
An HTML developer, also known as a front-end developer or web developer, specializes in creating and implementing the structure and layout of websites using HTML (Hypertext Markup Language).
JAVA Developer (2012-2014)
A Java developer is a professional who specializes in using the Java programming language to develop software applications, web applications, and other digital solutions.
PYTHON Developer (2014-2016)
A Python developer is a professional who specializes in using the Python programming language to build software applications, web applications, scripts, and data analysis tools.
NODE.JS Developer (2016-2018)
A Node.js developer is a specialized software engineer who primarily works with Node.js, a runtime environment that allows JavaScript to be used for server-side scripting.
